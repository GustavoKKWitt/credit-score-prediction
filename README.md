# credit-score-prediction
Machine learning project to predict whether a person's credit score is good or bad using different models and comparing their performance.

The goal is to test different machine learning models and compare their performance.

---

# 🎯 Objective

Predict the credit risk of customers using machine learning models and evaluate which algorithm performs better.

---

# 🧠 Technologies Used

- Python
- Pandas
- Scikit-learn
- Machine Learning

---

# 🤖 Models Tested

Two machine learning models were tested:

## Decision Tree
Accuracy: **83%**

Decision Trees performed better in this dataset because they can handle structured tabular data efficiently.

## Recurrent Neural Network (RNN)
Accuracy: **72%**

RNN was tested to compare neural networks with traditional machine learning models.

---

# 📊 Dataset

The dataset contains customer financial information used to predict credit risk.

Example features:

- age
- income
- payment history
- credit usage

Target variable:

- good credit score
- bad credit score
